## 📦 Installation

1.Clone the repo and cd into it
```
git clone https://github.com/Hello-May/php-crawler.git
```

2.install package
```
npm install
```

```
composer require
```

3.complete the setup
```
cp .env.example .env

```


```
php artisan key:generate
```

4.Set your database credentials in your .env file
